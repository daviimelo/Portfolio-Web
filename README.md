# Portfólio Web - Davi Melo

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido para apresentar a minha trajetória, as minhas competências técnicas e os projetos em que tenho trabalhado.

## Sobre o Projeto

Este site foi construído utilizando tecnologias fundamentais de desenvolvimento web. O objetivo é ter um espaço centralizado para exibir o meu trabalho e facilitar o contacto profissional.

### Estrutura do Repositório
- `assets/`: Imagens e recursos visuais.
- `styles/`: Ficheiros CSS para a estilização das páginas.
- `templates/`: Estrutura de ficheiros HTML.

## Tecnologias Utilizadas no Portfólio
- HTML5
- CSS3
